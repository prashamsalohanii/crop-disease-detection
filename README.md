#Overview
Agriculture is the backbone of Nepal's economy, yet many farmers lose significant portions of their harvest to preventable diseases. This project uses Deep Learning (CNN) to provide an instant diagnosis of plant health via a simple leaf photo.

 #Key Features
1.Instant Diagnosis: Upload a photo of a plant leaf and get an immediate prediction.

2.High Accuracy: Built using a Convolutional Neural Network (CNN) trained on the PlantVillage dataset.

3.User-Friendly UI: Simple interface built with Streamlit for accessibility on mobile and desktop.

4.Actionable Insights: (Optional feature) Provides names of diseases and recommended treatments.

 #Tech Stack
 
1.Framework: TensorFlow / Keras

2.Web Interface: Streamlit

3.Image Processing: PIL (Pillow)

4.Language: Python 3.x
